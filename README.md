Out of the Loop:
A local multiplayer social deduction game running in the console.

Project Status:
Current Build: Java (CLI/Console)

In Progress: Porting to a web-based interface using JavaScript.

Game Rules
Setup: Enter the number of players and their names.

Assignment: Select a category and then the game selects a secret word.

Roles:
Civilians: Receive the secret word.

The Impostor: Receives message that he is the impostor and does not know the word.

Interrogation: Players take turns asking questions to deduce who knows the word and who doesn't.

Voting: Players vote to eliminate the suspected Impostor.

Technical Details
Language: Java

Interface: System Console (Scanner/System.out)

Limitations: The current Java version relies on manual pacing (no built-in timer).
